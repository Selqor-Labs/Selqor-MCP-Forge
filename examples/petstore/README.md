# Petstore Worked Example

This example was generated from the repo's existing Petstore dashboard artifacts and then compared against Selqor Forge's built-in naive baseline.

## Files

- `uasf.json`: normalized API surface
- `analysis-plan.json`: grouping analysis
- `tool-plan.json`: curated tool plan
- `forge.report.json`: curated quality score
- `baseline-tool-plan.json`: one-endpoint-per-tool baseline
- `baseline-quality.json`: baseline score
- `summary.json`: concise comparison summary
- `curated-vs-baseline.svg`: chart used in the main README

## Comparison Snapshot

- Endpoints: 20
- Curated tools: 6
- Baseline tools: 20
- Curated score: 100
- Baseline score: 65
- Compression ratio: 0.25x curated vs 1.00x baseline
